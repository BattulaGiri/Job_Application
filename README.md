# Job Application Portal (Frontend Project)

This project is a simple **Job Application Form** built using **HTML, CSS, and JavaScript**.  
It demonstrates different HTML input types such as text, password, email, number, radio buttons, checkboxes, file uploads, date inputs, and more.

The form includes multiple "register" and "reset" buttons to help users test input behaviors.
